# 블로그 조회 및 최근 검색어 조회 API

### Using open source

* querydsl-jpa : 동적 쿼리 및 타입 안전하게 구현
* resilience4j-retry : 재시도 및 fallbacck 처리
* resilience4j-circuitbreaker : 문제를 감지하고 장애가 전파되는 것을 막기 위해 사용
* lombok : 반복되는 코드를 줄이기 위해
